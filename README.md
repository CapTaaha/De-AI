## 🎯 Overview

**De-AI** is a machine learning-based application designed to detect whether source code has been generated by AI models (such as ChatGPT, Copilot, etc.) or written by humans. This tool is valuable for:

- **Educational Institutions**: Ensuring academic integrity in programming assignments
- **Developers**: Identifying potential security or quality issues in AI-generated code
- **Code Review Teams**: Flagging AI-generated contributions for additional scrutiny
- **Researchers**: Studying patterns in human vs. AI-generated code

The system analyzes various code characteristics including complexity metrics, syntax patterns, naming conventions, and code structure to make intelligent predictions.

---

## ✨ Features

- 🔍 **Accurate Detection**: Machine learning model trained to distinguish between human and AI-generated code
- 📊 **Multiple Code Analysis Metrics**: 
  - Cyclomatic complexity
  - Code structure analysis
  - Naming convention patterns
  - Syntax pattern recognition
- 🎓 **Jupyter Notebook Implementation**: Easy to understand and modify
- 📈 **Comprehensive Analysis**: Detailed reports on code characteristics
- 🔐 **Privacy-Focused**: Analyze code locally without external API calls
- 📚 **Research-Backed**: Based on academic research and papers

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)
- Jupyter Notebook
- Virtual environment (recommended)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/CapTaaha/De-AI.git
cd De-AI
```

2. **Create a virtual environment** (recommended)
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. **Install dependencies**
```bash
pip install -r requirements.txt
```

4. **Launch Jupyter Notebook**
```bash
jupyter notebook
```

## 💻 Usage

### Running the Application

1. Open Jupyter Notebook in your browser
2. Navigate to the `src/` directory
3. Open the main notebook file and run the cells sequentially
4. Follow the presentation in the project root for detailed walkthrough

### Example Usage

```python
# Load the model and analyze code
from de_ai.detector import CodeDetector

detector = CodeDetector()
code_sample = """
def calculate_sum(numbers):
    total = 0
    for num in numbers:
        total += num
    return total
"""

result = detector.analyze(code_sample)
print(f"AI Generated: {result['is_ai_generated']}")
print(f"Confidence: {result['confidence']:.2%}")
```
